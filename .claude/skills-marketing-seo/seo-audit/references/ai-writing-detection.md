# AI Writing Detection Patterns

## Common AI Tells to Avoid

### Punctuation & Structure
- **Em dashes (—)** overused as connectors where commas or periods work
- **Colon-introduced lists** in every paragraph
- **Parallel structure obsession** — every list item perfectly balanced
- **Formulaic transitions** between sections

### Overused Phrases
- "In today's digital landscape"
- "It's worth noting that"
- "At the end of the day"
- "Let's dive in" / "Let's explore"
- "This comprehensive guide"
- "Whether you're a... or a..."
- "In this article, we'll cover"
- "Without further ado"
- "Harness the power of"
- "Navigate the complexities of"
- "Unlock the potential"
- "Elevate your"
- "Streamline your"
- "Leverage" (as a verb, excessively)
- "Delve into"
- "Tapestry" / "Landscape" / "Realm"

### Sentence Patterns
- Opening with gerunds ("By implementing...", "By leveraging...")
- Ending paragraphs with rhetorical questions
- Lists of exactly 3 items with identical grammatical structure
- Overly smooth transitions with no personality
- Every paragraph exactly 3-4 sentences
- Conclusions that summarize every section point

### Vocabulary Signals
- **Hedging everywhere**: "arguably", "perhaps", "it could be said"
- **False precision**: "a myriad of", "a plethora of"
- **Corporate bland**: "robust", "seamless", "cutting-edge", "game-changer"
- **Filler adjectives**: "invaluable", "indispensable", "crucial"

## How to Humanize

1. **Vary rhythm** — mix short punchy sentences with longer ones
2. **Use contractions** — "don't" not "do not", "it's" not "it is"
3. **Add opinion** — take a stance, be specific about preferences
4. **Include imperfection** — fragments, parenthetical asides, informal language
5. **Use concrete examples** — real tools, real numbers, real situations
6. **Write asymmetric lists** — items don't need identical structure
7. **Start with the point** — skip the throat-clearing intro
8. **Use "you" and "I"** — direct address over passive voice

## Detection Tools Reference

| Tool | Detection Method | Reliability |
|------|-----------------|-------------|
| Originality.ai | ML classifier | Moderate-high |
| GPTZero | Perplexity + burstiness | Moderate |
| Copyleaks | Neural network | Moderate |
| Manual review | Pattern matching | Highest |

Manual review by an experienced editor remains the most reliable detection method. Automated tools have high false-positive rates, especially with edited AI content.
